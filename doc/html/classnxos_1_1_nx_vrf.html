◆ getEvent()

virtual nxos::event_type_e nxos::NxVrf::getEvent ( )
pure virtual

To get the type of Event(Add, Update, Delete etc) when there is a change to a tracked VRF. Should be used only in Rib Handler Callbacks like postVrfCb(). If used outside of RIB Handler Callbacks its set to NO_EVENT.

Returns
Type of VRF Event(event_type_e)
C++:
class myRibMgrMgrHandler : public NxRibMgrHandler {
public:
bool postVrfCb(NxVrf *vrf) {
cout << "VRF Event: " << vrf->getEvent();
}
};
Python:
class pyRibHandler(nx_sdk_py.NxRibMgrHandler):
### Overloaded Callback to get VRF events
def postVrfCb(self, nxvrf):
print "VRF Event: %d" % vrf.getEvent()

To get the Table ID of the VRF for the given Address Family (AF) from the given VRF object. Returns 0 if the Table is not yet created.
