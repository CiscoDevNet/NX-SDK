◆ watchSyslog()

virtual bool nxos::NxTrace::watchSyslog ( bool  download = true)
pure virtual

Watch All syslog and get notified when a syslog occurs. User must have postSyslogCb registered to receive events. NOTE: watchSyslog does not work if watchOcPath in NxOcMgr for syslog is enabled.

Parameters
[in]download[Optional] true or false. If true, all syslogs already created in the system will be notified via postSyslogCb and the end of download is signalled by overloading the respective postSyslogDownloadDone callback.
Returns
bool True if the watch is successful. False if its unsuccessful.
Since
NXSDK V2.5
C++:
#include <nx_sdk.h>
#include <nx_trace.h>
NxTrace *trace;
sdk = nxos::NxSdk::getSdkInst(argc, argv);
trace = sdk->getTracer();
if (trace) trace->watchSyslog();
Python:
import nx_sdk_py
sdk = nx_sdk_py.NxSdk.getSdkInst(len(sys.argv), sys.argv)

trace.watchSyslog()
