Detailed Description

Specifies that the type of the CLI input parameter is an integer.

Matching param type is P_INTEGER. When used in updateParam context:

Parameters
[in]min_valminimum allowed value for the input interger parameter.
[in]max_valMaximum allowed value for the input interger parameter.

Usage: If you need to create a CLI which will take an integer between <1-100>

CLI: $appname port-bw threshold ? <1-100> Threshold Limit. Default 50%

sdk = nxos::NxSdk::getSdkInst(argc, argv);
cliP = sdk->getCliParser();
nxos::cli_param_type_integer_attr int_attr = {0};
int_attr.min_val = 1;
int_attr.max_val = 100;
cmd = cliP->newConfigCmd("set_port_bw_threshold_cmd",
"port-bw threshold <threshold>");
cmd->updateParam("<threshold>", "Threshold Limit. Default 50%", nxos::P_INTEGER,
(void *)&int_attr, sizeof(int_attr));
