typedef struct nxos::cli_param_type_integer_attr_ nxos::cli_param_type_integer_attr

Specifies that the type of the CLI input parameter is an integer.

Matching param type is P_INTEGER. When used in updateParam context:

Parameters
[in]min_valminimum allowed value for the input interger parameter.
[in]max_valMaximum allowed value for the input interger parameter.

Usage: If you need to create a CLI which will take an integer between <1-100>

CLI: $appname port-bw threshold ? <1-100> Threshold Limit. Default 50%

sdk = nxos::NxSdk::getSdkInst(argc, argv);
cliP = sdk->getCliParser();
nxos::cli_param_type_integer_attr int_attr = {0};
int_attr.min_val = 1;
int_attr.max_val = 100;
cmd = cliP->newConfigCmd("set_port_bw_threshold_cmd",
"port-bw threshold <threshold>");
cmd->updateParam("<threshold>", "Threshold Limit. Default 50%", nxos::P_INTEGER,
(void *)&int_attr, sizeof(int_attr));
import nx_sdk_py
sdk = nx_sdk_py.NxSdk.getSdkInst(len(sys.argv), sys.argv)
cliP = sdk.getCliParser()
int_attr = nx_sdk_py.cli_param_type_integer_attr()
int_attr.min_val = 1;
int_attr.max_val = 100;
cmd = cliP.newConfigCmd("set_port_bw_threshold_cmd",
"port-bw threshold <threshold>")
cmd.updateParam("<threshold>", "Threshold Limit. Default 50%", nxos::P_INTEGER,
int_attr, len(int_attr))

typedef struct nxos::cli_param_type_ip_addr_attr_ nxos::cli_param_type_ip_addr_attr

Specifies that the type of the CLI input parameter is ipv4 or ipv6 address or prefix.

Matching param type is P_IP_ADDR. When used in updateParam context:

Parameters
[in]is_ipv6If set then Input parameter is a IPv6 addr else its IPv4 address.
[in]is_maskIf set then Input parameter is a Prefix.

Usage: If you need to create a CLI which will take a IP or IPv6 Prefix 

CLI: $appname filter-prefix ?
     A.B.C.D/LEN        Enter Filter IP Prefix
     A:B::C:D/LEN       Enter Filter IPv6 Prefix
sdk = nxos::NxSdk::getSdkInst(argc, argv);
cliP = sdk->getCliParser();
nxos::cli_param_type_ip_addr_attr ip_pfx_attr = {0};
ip_pfx_attr.is_ipv6 = false;
ip_pfx_attr.is_prefix = true;
cmd = cliP->newConfigCmd("set_filter_prefix_cmd",
"filter-prefix [<ip-prefix> | <ipv6-prefix>]");
cmd->updateParam("<ip-prefix>", "Enter Filter IP Prefix", nxos::P_IP_ADDR,
(void *)&ip_pfx_attr, sizeof(ip_pfx_attr));
ip_pfx_attr.is_ipv6 = true;
cmd->updateParam("<ipv6-prefix>", "Enter Filter IPv6 Prefix", nxos::P_IP_ADDR,
(void *)&ip_pfx_attr, sizeof(ip_pfx_attr));
import nx_sdk_py
sdk = nx_sdk_py.NxSdk.getSdkInst(len(sys.argv), sys.argv)
cliP = sdk.getCliParser()
ip_pfx_attr = nx_sdk_py.cli_param_type_ip_addr_attr()
ip_pfx_attr.is_ipv6 = False
ip_pfx_attr.is_prefix = True
cmd = cliP.newConfigCmd("set_filter_prefix_cmd",
"filter-prefix [<ip-prefix> | <ipv6-prefix>]")
cmd.updateParam("<ip-prefix>", "Enter Filter IP Prefix", nxos::P_IP_ADDR,
ip_pfx_attr, len(ip_pfx_attr))
ip_pfx_attr.is_ipv6 = True
cmd.updateParam("<ipv6-prefix>", "Enter Filter IPv6 Prefix", nxos::P_IP_ADDR,
ip_pfx_attr, len(ip_pfx_attr));

◆ param_type_e

enum nxos::param_type_e

Supported Input Parameter Types in a CLI.

Types help/guide the user to input the right information for a CLI thereby validating the input in the CLI parser itself rather than validating the CLI config in the backend callback

Each Type has some optional additional attributes to facilitate the CLI input to be taken for a parameter. Refer to updateParam.

P_INTEGER | cli_param_type_integer_attr P_STRING | cli_param_type_string_attr P_INTERFACE | NULL P_IP_ADDR | cli_param_type_ip_addr_attr P_MAC_ADDR | NULL P_VRF | NULL
