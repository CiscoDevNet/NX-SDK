virtual bool nxos::NxAdjMgr::discoverAdj ( const std::string &  intfName,
const std::string &  ipAddr 
)
pure virtual

Discover an adjacency by probing the interface and IP address for an adjacency. If one exists, user will be notified via postAdjCb. User must have postAdjCb registered and have a watch on the host via watchAllAdjs or watchAdj API.

Parameters
[in]intfNameinterface name.
[in]ipAddrIPV4 or IPV6 Address.
Returns
true if discoverAdj operation is successful
C++:
#include <nx_sdk.h>
#include <nx_adj_mgr.h>
NxAdj *adj;
sdk = nxos::NxSdk::getSdkInst(argc, argv);
adjMgr = sdk->getAdjMgr();
adjMgr->watchAllAdjs();
adj = adjMgr->discoverAdj("Vlan100", "10.1.1.2");
Python:
import nx_sdk_py
sdk = nx_sdk_py.NxSdk.getSdkInst(len(sys.argv), sys.argv)

Watch an adjacency

Parameters
[in]intfNameinterface name in string.
[in]addr_family[Optional] nxos::AF_IPV4 or nxos::AF_IPV6
[in]download[Optional] true or false. If true, if the adjacency is already created in the system will be notified via postAdjCb and the end of download is signalled by overloading the respective postAdjIPv4DownloadDone (or) postAdjIPv6DownloadDone callbacks.
[in]ipAddr[Optional] IP address in string. If not specified, "" is used to watch on all IP address.
Returns
bool True - if successful. False - if not successful.
Exceptions
ERR_INVALIDif intfName or addr_family or ipAddr is invalid.

Watch All Adjacencies and get notified when an adjacency is created or deleted. User must have postAdjCb registered to receive events.
