Detailed Description

Public interface to NX-SDK interface events.

The application should derive its own handler class from this and override the methods. If a watch is set for an interface, these events will be delivered to the application if the method is provided in the application handler class. The interface object passed to the handler is owned by the library and should not be stored or freed by the application. Interface Mgr events generated from NXOS are delivered to application callbacks through NXSDK interface objects Steps to register for interface events: 1) Open NXSDK interface mgr object using getIntfMgr() 2) Define application callbacks by re-defining NxIntfMgrHandler class 3) Install application handlers using setIntfHandler() 4) call NxSdk::startEventLoop()
