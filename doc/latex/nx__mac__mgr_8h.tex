\subsection{Detailed Description}
Abstract S\+DK interface to Nexus(\+N\+X\+O\+S) M\+AC Management. 

Provides necessary Abstraction/\+Plugin for N\+X\+OS M\+AC functionalities. It provides the Ability to register for M\+AC updates and more.

Steps to register for M\+AC event\+: 1) Instantiate Nx\+S\+DK Mac Manager Object using get\+Mac\+Mgr(). 2) To watch for M\+AC updates, use watch\+Mac() (or) watch\+All\+Mac() for all M\+AC updates. 3) Register for callback handler with Nx\+Mac\+Mgr to get the regsitered M\+AC updates. 4) Overload Nx\+Mac\+Mgr\+Handler class and its member functions like (post\+Mac\+Cb() -\/ To get watched M\+AC updates) with your own implementation to handle various events.
