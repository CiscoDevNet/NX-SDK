\doxysubsection{Detailed Description}
Abstract Exception Interface for all thrown NX errors. 

N\+X\+S\+DK V1.\+0 supports
\begin{DoxyItemize}
\item Throws Simple exception of the type in C++ -\/ std\+::logic\+\_\+err Python -\/ String Object.
\end{DoxyItemize}

N\+X\+S\+DK V1.\+5 supports
\begin{DoxyItemize}
\item Throws Simple exception of the type as version V1.\+0, C++ -\/ std\+::logic\+\_\+err Python -\/ String Object.
\item Also Throws Advanced exeception of the type Nx\+Exception across all languages with more information. ~\newline
 C++ -\/ Nx\+Exception Object Python -\/ Nx\+Exception Object
\end{DoxyItemize}

\begin{DoxyNote}{Note}
By default, simple exception is enabled for an App. If the App supports Advanced Exception handling then to enable Advanced exception, set adv\+Exception flag to T\+R\+UE in get\+Sdk\+Inst Api.
\end{DoxyNote}
